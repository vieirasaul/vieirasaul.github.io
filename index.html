Sobre o site

Esse site foi construido responsivamente com HTML5 e CSS3, e também Javascript e Jquery. As fontes utilizadas são distribuidas pelo Google Fonts e os ícones vem do Font-Awesome. O framework utilizado para a escrita animada no banner foi o TypeIt. O versionamento foi controlado com Git pelo GitHub.
